Frequently Asked Questions

Do I need API keys to use Love Llama?

The Red Flag Checklist works without any API keys. For AI-powered chat analysis, you need an Anthropic API key. For photo verification, you need a Reality Defender API key. Both can be obtained from their respective websites.

Where can I get API keys?

  • Anthropic: console.anthropic.com
  • Reality Defender: app.realitydefender.ai

Is my data safe?

Yes. All data is stored locally on your device with iOS encryption. API keys are stored in the secure Keychain. We don't collect analytics or track your usage. See our Privacy Policy for details.

How do I delete my data?

Go to Settings → Data & Privacy → Delete All Data. This removes all conversations, photo checks, checklist sessions, and API keys. Alternatively, uninstalling the app removes all local data.

Does chat analysis work without internet?

Yes! Love Llama includes a local pattern scanner with 300+ romance scam indicators that works entirely offline. The AI-powered deep analysis via Claude requires internet, but the local scanner provides instant results regardless.
