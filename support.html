Frequently Asked Questions

Do I need API keys to use Love Llama?

No! All features work fully offline without any API keys. Chat analysis uses a built-in local pattern scanner, photo verification uses on-device image analysis via Apple's Vision framework, and the Red Flag Checklist is entirely local. Adding API keys for Anthropic (chat) and Reality Defender (photo) enhances accuracy with AI-powered analysis, but they're completely optional.

Where can I get API keys?

  • Anthropic: console.anthropic.com
  • Reality Defender: app.realitydefender.ai

Is my data safe?

Yes. All data is stored locally on your device with iOS encryption. API keys are stored in the secure Keychain. We don't collect analytics or track your usage. See our Privacy Policy for details.

How do I delete my data?

Go to Settings → Data & Privacy → Delete All Data. This removes all conversations, photo checks, checklist sessions, and API keys. Alternatively, uninstalling the app removes all local data.

Does Love Llama work without internet?

Yes! All features work fully offline. Chat analysis uses a local pattern scanner with 300+ romance scam indicators. Photo verification uses on-device image analysis powered by Apple's Vision framework. The Red Flag Checklist is entirely local. Adding optional API keys enhances results with AI, but everything works without them.

What data is shared with third parties?

Only if you add an API key and explicitly approve sharing at the point of use. Chat text can be sent to Anthropic's Claude API, and photos can be sent to Reality Defender's API. You're always asked before any data leaves your device, and you can choose "Analyze Locally Only" instead. See our Privacy Policy for full details.
