Privacy Policy

Last updated: February 21, 2026

Love Llama is committed to protecting your privacy. This policy explains what data we collect, how it's used, who it's shared with, and your rights.

1. Data We Collect

Love Llama processes the following data locally on your device:

  • Conversation text you paste or scan via screenshots
  • Photos you select for AI-generated image detection
  • Red flag checklist selections
  • Analysis history and results
All data is stored on-device using encrypted storage (iOS Data Protection). Love Llama does not collect, store, or transmit data to any server we operate.

2. How We Collect Data

Data is collected only through your direct actions within the app:

  • Text input: You manually paste or type conversation text, or select a screenshot for OCR text extraction (performed on-device using Apple's Vision framework)
  • Photo selection: You manually select a photo from your photo library using Apple's PhotosPicker
  • Checklist input: You manually toggle checklist items

No data is collected automatically, in the background, or without your direct initiation.

3. Third-Party AI Services — Data Sharing

Love Llama optionally integrates with two third-party AI services. Data is only shared with these services when (a) you have added an API key for that service, and (b) you have given explicit in-app permission to share data with that specific service. You are prompted for permission before any data leaves your device.

Chat Analysis — Anthropic (Claude API)

  • What is sent: The conversation text you entered or extracted from a screenshot. No personal identifiers, contact names, platform names, or device information are included.
  • Where it is sent: Anthropic's Claude Messages API at api.anthropic.com
  • Who receives it: Anthropic, PBC (San Francisco, CA)
  • Why: To analyze conversation text for romance scam patterns using AI
  • Their policy: anthropic.com/policies/privacy
  • Without an API key: Analysis is performed entirely on-device using local pattern matching. No data is sent anywhere.

Photo Verification — Reality Defender

  • What is sent: The photo image you selected. Image EXIF metadata is stripped. No file names, personal information, or device information are included.
  • Where it is sent: Reality Defender's API at api.prd.realitydefender.xyz
  • Who receives it: Reality Defender, Inc. (New York, NY)
  • Why: To detect whether a profile photo was generated by AI
  • Their policy: realitydefender.com/privacy-policy
  • Without an API key: Analysis is performed entirely on-device using Apple's Vision framework and image heuristics. No data is sent anywhere.

Both third-party services process data according to their own privacy policies. Anthropic and Reality Defender both provide protections for user data, including not using API data to train their models.

4. User Consent & Permissions

Love Llama requires your explicit permission before sharing data:

  • On first launch, a consent screen explains all third-party data sharing
  • Before the first time data is sent to each third-party service, a specific alert explains exactly what data will be sent and to whom
  • You can choose "Analyze Locally Only" at any prompt to keep all data on-device
  • You can revoke all data processing consent at any time in Settings
  • Revoking consent immediately stops all third-party data sharing
All features work without any API keys using on-device analysis only.

5. API Keys

Your API keys for Anthropic and Reality Defender are stored securely in the iOS Keychain with hardware-level encryption (kSecAttrAccessibleWhenUnlockedThisDeviceOnly). Keys are only used to authenticate requests to their respective services and are never sent to our servers or shared with any other party.

6. Data Storage & Security

  • All analysis history is stored locally on your device only
  • Data is encrypted at rest using NSFileProtectionComplete
  • No data is backed up to our servers (no servers exist)
  • You can delete all data at any time from Settings
  • Photos stored in history are saved as compressed JPEG data on-device only

7. No Tracking or Analytics

Love Llama does not:

  • Track your activity or behavior
  • Collect analytics, crash reports, or usage data
  • Use advertising identifiers (IDFA)
  • Share data with advertisers or data brokers
  • Use cookies, tracking pixels, or fingerprinting
  • Contain any third-party analytics SDKs

8. Your Rights

You have full control over your data:

  • Delete all stored data from Settings at any time
  • Remove API keys at any time to stop all third-party sharing
  • Revoke data processing consent at any time
  • Uninstall the app to remove all local data
  • Use all features in local-only mode without sharing any data

9. Children's Privacy

Love Llama is not intended for children under 13. We do not knowingly collect data from children.

10. Changes to This Policy

If we update this privacy policy, we will update the "Last updated" date above. Continued use of the app after changes constitutes acceptance of the revised policy.
